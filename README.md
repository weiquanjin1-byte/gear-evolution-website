# gear-evolution-website

“小齿轮进化中”的个人网页项目，用来展示 AI 学习成长记录、机械工程背景、作品集和未来 AI + 机械方向的探索。

## 项目定位

这是一个面向个人成长记录的单页网站。视觉方向是赛博朋克、机械工程与 AI 科技感结合：深色背景、霓虹蓝紫、HUD 线条、发光齿轮和半透明信息卡片。

## 技术栈

- React
- Vite
- Tailwind CSS
- lucide-react

## 页面模块

- Hero 首页主视觉
- 关于我 / About Me
- AI 学习日志 / AI Learning Journey
- 机械工程背景 / Mechanical Engineering
- 项目作品集 / Projects
- 联系方式 / Contact

## 本地运行

```bash
npm install
npm run dev
```

构建生产版本：

```bash
npm run build
```
